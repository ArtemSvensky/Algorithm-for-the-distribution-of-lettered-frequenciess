Приложение для распределения литерных частот МРЛС на ведущей БМ между БМ батареи

Это консольное приложение, написанное на C++, которое выполняет распределение литерных частот МРЛС на "ведущей" БМ между БМ батареи согласно указанной логике.

Как использовать

Введите в консоль количество БМ в батарее.
Для каждой БМ введите следующие данные:

	- Идентификатор БМ
	- Координата X
	- Координата Y
	- Количество исполнений РПДС МРЛС для данной БМ

После ввода данных приложение выполнит распределение литерных частот МРЛС на ведущей БМ между БМ батареи и выведет результат на экран.

Пример:

Введите количество БМ в батарее: 6
Введите данные для каждой БМ:
	1 2.0 5.0 1
	2 3.0 7.0 2
	3 4.0 2.0 1
	4 5.0 9.0 2
	5 6.0 3.0 1
	6 7.0 2.0 2

Результат распределения частот МРЛС:

	BM1 A
	BM2 B
	BM3 C
	BM4 A
	BM5 B
	BM6 C
